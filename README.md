# Blinkit-Data-Analysis-
This project performs end-to-end exploratory data analysis on Blinkit sales data using Python. It analyzes sales performance, outlet behavior, product category trends, and customer ratings. The project includes data cleaning, KPI generation, and visualizations to deliver actionable business insights for data-driven decision-making.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Handling missing and inconsistent values
- KPI calculation and aggregation
- Outlet-wise, product-wise, and location-wise analysis
- Visualization of sales trends and performance metrics

## Key Insights
- Supermarket Type 1 contributes the highest revenue among all outlet types.
- Fruits and Vegetables category generates the maximum sales.
- Tier 3 outlets show strong sales potential, indicating growth opportunities in semi-urban areas.
- Outlet age has a positive impact on sales performance.
- Customer ratings mostly fall between 3.5 and 4.5, indicating stable customer satisfaction.

## Business Recommendations
- Expand Supermarket Type 1 outlets in high-performing Tier 3 locations.
- Increase inventory for high-performing product categories like Fruits and Vegetables.
- Improve product visibility for low-selling items to boost sales.
- Focus marketing efforts on newer outlets to accelerate performance growth.

## Conclusion
This project demonstrates practical Python data analysis skills, including EDA, KPI generation, and visualization. It highlights the ability to convert raw business data into actionable insights, making it suitable for a Python Data Analyst portfolio project.
